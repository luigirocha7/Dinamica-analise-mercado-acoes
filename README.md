# 📊 Análise de Mercado de Ações com Tabela Dinâmica

Este projeto tem como objetivo analisar operações de compra e venda de ações utilizando tabelas dinâmicas no Google Sheets.

## 📁 Estrutura do Projeto

* `compra_venda`: Quantidade e valor total de operações de compra e venda
* `cliente_compra_venda`: Análise por cliente
* `cliente_lucro_prejuizo`: Lucro/prejuízo por ativo (ticker)

## 📌 Metodologia

O lucro foi calculado manualmente com base na fórmula:

Lucro = Venda - Compra

As tabelas dinâmicas foram utilizadas para organizar e resumir os dados.
